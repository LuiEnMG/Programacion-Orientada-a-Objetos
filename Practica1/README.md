Instalacion de Apache Netbeans y Java JDK

Apache Netbeans (Windows):
1. Ir a la pagina https://netbeans.apache.org/front/main/download
2. Dar click en Download para la version Apache NetBeans 27
3. Seleccionar la version a descargar, bajar a Installers and Packages
4. Dar cick en "Friends of Apache Netbeans"
5. Dar click en la opcion de "Windows (x86_64)" debajo de Download
6. Esperar a que se descargue y abrir el instalador
7. Seguir las instrucciones que el instalador pide

Java JDK (Windows):
1. Ir a la pagina https://www.oracle.com/java/technologies/downloads
2. En la opcion de JDK 25, seleccionar los paquetes para Windows
3. Dar click en el enlace enfrente de la opcion de "x64 Installer"
4. Seguir las instrucciones del instalador, cambiar el directorio de la instalacion si es deseado
